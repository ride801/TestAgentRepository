# Catan Game Assets

This directory contains game assets like images, icons, and other media files.

## Suggested Assets to Add

- Favicon (favicon.ico)
- Resource card images
- Development card images
- Board textures
- Player tokens
- Sound effects (dice rolls, building sounds)
- Background music (optional)

For now, the game uses emoji icons for resources:
- 🌲 Wood
- 🧱 Brick
- 🐑 Sheep
- 🌾 Wheat
- ⛰️ Ore
